This is a project of a Taverna Workbench plugin. 
The main purpose of this plugin is adding a new saving logic feature and modifing the user interface of Taverna Workbench to use this feature.
Specifically this plugin allows to save a workflow in an online database adding to an existing dataset. This operation is made thruogh SSHOC dataverse tool.
SSHOC dataverse tool is a web service that provide support to comunicate with Dataverse.
Dataverse is an open sourche web application to store and manage research data (repository).

Actually to use this functionality with Taverna Workbench is needed to download the source code compiling it with java and execute Taverna.
The functionality that will make it a downloadable plugin will be implemented soon.
